JavaScript Tip Dönüşümleri

Sayıları Dizelere Dönüştürme

Global yöntem String()sayıları dizelere dönüştürebilir. Herhangi bir sayı, değişmez değer, değişken veya ifadede kullanılabilir:

                    String(x)        
                    String(123)      
                    String(100 + 23)

Booleanları Dizelere Dönüştürme

Global yöntem String()boolean'ları dizelere dönüştürebilir.

                String(false)      // returns "false"
                String(true)       // returns "true"

Tarihleri ​​Dizelere Dönüştürme

Global yöntem String()tarihleri ​​dizelere dönüştürebilir.

                String(Date())  // returns "Thu Jul 17 2014 15:38:19 GMT+0200 (W. Europe Daylight Time)"

Dizeleri Sayılara Dönüştürme

Global yöntem Number(), dizeleri sayılara dönüştürebilir. Sayı içeren dizeler ("3.14" gibi) sayılara dönüştürülür (3.14 gibi). Boş dizeler 0'a dönüşür. Başka bir şey NaN(Sayı Değil) biçimine dönüşür .

                Number("3.14")    // returns 3.14
                Number(" ")       // returns 0
                Number("")        // returns 0
                Number("99 88")   // returns NaN

Booleanları Sayılara Dönüştürme

Global yöntem Number()boolean'ları sayılara dönüştürebilir.

                Number(false)     // returns 0
                Number(true)      // returns 1

Tarihleri ​​Sayılara Dönüştürme

Genel yöntem Number(), tarihleri ​​sayılara dönüştürmek için kullanılabilir.

                d = new Date();
                Number(d)          // returns 1404568027739
