JavaScript Yazılım (Syntax) Yapısı

JavaScript Syntax, JavaScript programlarının nasıl oluşturulduğu ile ilgili kurallar dizisidir:

                var x, y, z;       
                x = 5; y = 6;     
                z = x + y;

JavaScript Değerleri

JavaScript sözdizimi iki tür değer tanımlar: Sabit değerler ve değişken değerler. Sabit değerlere değişmez değerler denir . Değişken değerlerine değişken denir .

JavaScript Değişmezleri

Sabit değerler yazmak için en önemli kurallar şunlardır:

  • Sayılar ondalıklı veya ondalıksız olarak yazılır: 
           10.50

       1001
  • Dizeler çift ​​veya tek tırnak içinde yazılan metindir: 
            "John Doe"

       'John Doe'

JavaScript Değişkenleri

Bir programlama dilinde, veri değerlerini saklamak için değişkenler kullanılır . JavaScript, değişkenleri bildirmek için varanahtar kelimeyi kullanır . Bir eşittir işareti kullanılır atama değerlere değişkenlere. Bu örnekte, x bir değişken olarak tanımlanmıştır. Sonra, x değeri 6'ya atanır (verilir): 

            var x;

            x = 6;

Not: Tüm JavaScript tanımlayıcıları büyük / küçük harfe duyarlıdır . Değişkenler lastNameve lastname, iki farklı değişkendir:

            var lastname, lastName;
            lastName = "Doe";
            lastname = "Peterson";
